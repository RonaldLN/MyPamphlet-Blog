    pyrealsense2/__init__.py

from .pyrealsense2 import *

    而 __init__.py 路径下是 pyrealsense2.cp310-win_amd64.pyd 文件，就意识到这是一个使用c++编写的库(而python只是将其进行了封装)，所以无法直接从源代码中查看库中的类和函数。
